# 🌌 Coder_Syndicate

> **"Turning concepts into code, and code into impact."**

**Coder_Syndicate** is a next-generation developer platform that bridges the gap between *learning computer science* and *applying it in real-world scenarios*. It transforms traditional documentation into an **interactive, competitive, and social coding ecosystem**.

---

## ✨ Core Features

### ⚔️ The Arena (Competitive Coding)

* **Quick Match**
  Get instantly matched in real-time 1v1 coding battles with developers of similar skill levels.

* **Friendly Match**
  Challenge your friends directly with custom battle invites.

* **Community Battles**
  Join global syndicates, form squads, and compete against rival teams.

---

### 🤖 XSyndicate AI Assistant

* Instantly explain complex concepts by highlighting documentation text.
* Personalized, context-aware AI responses for better learning.
* **Voice Integration**
  * Real-time voice conversations
  * High-quality Text-to-Speech for immersive tutoring

---

### 🤝 Global Syndicate (Social System)

* Add and manage friends within your developer network.
* Real-time online/offline presence tracking.
* Live notifications for match requests and interactions.

---

### 🏆 Progression & Leaderboards

* Earn XP through learning and competitive matches.
* Rank up through tiers (e.g., Bronze → Platinum → Diamond).
* Compete globally and track your leaderboard position.

---

### 💎 Premium UI/UX

* Modern **Light Glassmorphism** design
* Smooth animations and gradient effects
* Clean, responsive, and interactive interface

---

## 🛠️ Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS (Styling & Animations)
* React Router DOM (Navigation)

### Backend & Authentication

* Node.js & Express.js (REST API)
* Firebase Authentication (Google & Email login)

### AI & Voice Integration

* Google Gemini API (AI explanations & chat)
* ElevenLabs API (Text-to-Speech)

---

## 🚀 Getting Started

### 📋 Prerequisites

Make sure you have:

* Node.js (v16 or higher)
* npm or yarn
* Firebase project setup
* API keys for Gemini and ElevenLabs
